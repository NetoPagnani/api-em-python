📚 API de Gerenciamento de Livros
Uma API REST simples e funcional desenvolvida em Python com o framework Flask para gerenciar um acervo de livros. Este projeto permite realizar as operações básicas de um CRUD (Create, Read, Update e Delete).

🚀 Tecnologias
- Python 3

- Flask: Micro-framework web.

- JSON: Formato para estruturação dos dados.
